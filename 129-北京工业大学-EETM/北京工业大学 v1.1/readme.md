# racecar_ws使用说明
####请按照先前教程完成ros的安装、工作空间的创建和仿真环境的搭建。

###1. 建立工作空间racecar_ws
进入start_game文件夹打开终端

**1.1 新建工作空间命名为racecar_ws,将压缩包中的gazebo_pkg,race_navigation,start_game功能包放在工作空间src下**
> * 请将start_plane、end_plane文件夹复制到.gazebo/models目录下

**1.2 编译工作空间**
> * 小车模型的部分参数已做修改以及robotBaseFrame已修改为dummy

### 2.运行赛事程序
> * 在终端输入如下指令：

```bash
python2 main.py       
```
